
Blender2.57でメタセコイアフォーマットを入出力するスクリプトです。
Windows XPで動作確認済みです。

・設置方法
「Blender2.57のインストールフォルダ」→「2.57」→[scripts」→「addons」にフォルダごと設置して、Blenderを再起動してください。


・注意
Blender2.57でのみ対応しています。下位バージョンでは動作しません。
曲面等には対応してません。
ミラーリングは、全て鏡面分離として処理されます。

詳しくはhttp://pr0jectze10.tuzigiri.com/simpleVC_20101030133458.htmlを参照してください。
